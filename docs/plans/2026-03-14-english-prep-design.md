# AI 영어 준비 시스템 설계

## 목표

7월 제주도 미국 대학생 합동 교육(3주)을 대비하여, 일상 영어 회화 + AI/ML 기술 영어를 커리큘럼과 연동해 준비할 수 있는 시스템.

## 아키텍처

`/english` 페이지에 3개 탭:

1. **상황별 표현 카드** — 일상/기술/제주도 특화 필수 영어 표현
2. **AI 영어 대화 연습** — 미국 학생 역할 시뮬레이션 + 교정 모드 토글
3. **오늘의 기술 영어** — 커리큘럼 연동 개념 영어 설명 연습

---

## 1. 상황별 표현 카드

카테고리별 필수 영어 표현:
- **일상**: 자기소개, 식사/카페, 일정 조율, 감사/부탁
- **기술**: 코드 설명, 모델 성능, 에러 논의, 아이디어 제안
- **제주도 특화**: 프로젝트 소개, 팀 토론, 발표, 소셜 상황

각 카드: 한국어 상황 → 영어 표현 2~3개 → 예시 대화. "이 표현 연습하기" → 대화 탭 이동.

---

## 2. AI 영어 대화 연습

- AI가 미국 대학생 역할로 영어 대화 시뮬레이션
- 시나리오 선택: 첫 만남, 코드 리뷰, 점심 대화, 프로젝트 소개 등
- 교정 모드 토글: OFF=대화만, ON=문법/표현 상세 피드백
- 커리큘럼 연동: 오늘 배운 주제로 기술 대화 시나리오 자동 생성

---

## 3. 오늘의 기술 영어

- 커리큘럼 연동 개념을 영어로 설명하는 연습
- 사용자 입력 → AI 교정 + 모범 답안
- 용어 사전(glossary) 연동

---

## 데이터 & API

- DB 불필요 (세션 내 state 관리)
- `POST /api/english/chat` — 대화 시뮬레이션 + 교정
- `POST /api/english/explain` — 기술 영어 설명 연습 + 교정
- AI 모델: `gpt-4o` (자연스러운 영어 + 교정 품질)
- 표현 카드: `src/data/english-phrases.ts` (정적 데이터)

---

## 컴포넌트 구조

- `src/app/english/page.tsx` — 메인 페이지 (3탭)
- `src/data/english-phrases.ts` — 상황별 표현 데이터
- `src/app/api/english/chat/route.ts` — 대화 API
- `src/app/api/english/explain/route.ts` — 기술 영어 API

## 기술 스택

- Next.js API Routes, OpenAI GPT-4o, ReactMarkdown, Tailwind CSS
