# AI 용어 코칭 + 용어 사전 설계

## 배경

비전공자 사용자가 AI/ML 용어의 개념과 활용을 체계적으로 학습할 수 있도록, AI 튜터에 용어 전문 코칭 모드를 추가하고 학습한 용어를 저장/반복 열람할 수 있는 용어 사전 페이지를 제공한다.

## 핵심 기능

### 1. AI 튜터 용어 모드 (탭 추가)

- `/tutor` 페이지 상단에 "일반 질문" / "용어 코칭" 탭 전환
- 용어 코칭 모드에서 질문하면 AI가 구조화된 형식으로 응답:
  - **개념 정의** — 비전공자 눈높이 설명
  - **실제 예시** — 코드/상황 예시
  - **활용법** — 어디서 어떻게 쓰이는지
  - **연관 용어** — 함께 알면 좋은 용어들
- 설명 후 자유롭게 대화 이어가기 가능
- 대화 중 설명된 용어는 자동으로 DB에 저장

### 2. 용어 사전 페이지 (`/glossary`)

- 사이드바에 "용어 사전" 메뉴 추가
- 모듈별 분류: 파이썬 마스터, 딥러닝 기초, CV 기초 등
- 검색 기능: 용어명으로 검색
- 용어 카드 클릭 시 저장된 AI 설명 보기
- "다시 대화하기" 버튼 → 튜터 용어 모드로 이동하여 추가 질문

### 3. DB 구조

- `glossary_terms` 테이블:
  - `id` (uuid, PK)
  - `user_id` (uuid, FK → auth.users)
  - `term` (text) — 용어명
  - `module` (text) — 소속 모듈
  - `definition` (text) — AI 설명 전체 (마크다운)
  - `related_terms` (text[]) — 연관 용어 배열
  - `created_at` (timestamptz)
  - `updated_at` (timestamptz)
- RLS: 본인 데이터만 CRUD

### 4. API

- `/api/tutor/glossary` POST — 용어 코칭 대화 (구조화된 시스템 프롬프트)
- `/api/glossary` GET — 저장된 용어 목록 조회 (모듈별 필터, 검색)
- `/api/glossary` POST — 용어 저장
- `/api/glossary/search` GET — 용어 검색

### 5. AI 프롬프트 전략

- 시스템 프롬프트에 구조화된 응답 형식 명시
- 모델: 용어 설명은 gpt-4o (깊은 이해 필요), 용어 추출/분류는 gpt-4o-mini
- AI 튜터와의 차별점: 자유 질문이 아닌 용어 중심 코칭

### 6. 향후 확장 (현재 미구현)

- 플래시카드 반복 학습 (간격 반복 알고리즘)
- 퀴즈 모드 (4지선다, 설명하기)
